
WHAT IS SPEECH.ZIP
---------------------

It's a set of voice samples required by Witnesscam's speech
sysnthesis system.


HOW TO USE SPEECH.ZIP
---------------------

To prepare a blank of freshly formatted SD-card for use with
the Witnesscam system, please extract the entire "speech"
folder from "speech.zip" and copy it -together with its
contents- to the SD-card's root.
